🔵 점검 순서

  1. 로그인 → 소속 구분·과·팀·이름 입력
  2. [현장 목록] 탭 → 내 담당 현장 확인 (파란 "내 담당" 태그)
  3. 현장 카드 클릭 → 점검표 팝업 오픈
  4. 침수방지시설 / 차수판 항목별 ✅ 적합 / ❌ 부적합 / ➖ 해당없음 선택
  5. 현장 사진 업로드 (드래그&드롭 가능)
  6. [점검 완료 저장] 클릭
  7. [결과 보고서] 탭 → 행 클릭 → 세부 팝업 확인 및 사진 다운로드

🟣 중복설치 현장 안내

  • 침수방지시설과 차수판이 모두 설치된 현장은 중복설치 태그로 표시
  • 점검표에 두 시설 항목이 모두 포함되어 한 번에 점검 가능

🔴 주의사항

  • 부적합 항목 발견 시 메모란에 사유 필수 기재
  • 사진은 시설 전체가 보이도록 촬영
  • 비 오는 날 스마트폰 방수 처리 후 사용
