🔵 점검 순서

  1. 로그인 → 소속 구분·과·팀·이름 입력
  2. [현장 목록] 탭 → 지역(동·읍·면) 카드 선택
  3. 현장 카드 클릭 → 점검표 팝업 오픈
  4. 침수방지시설 / 차수판 항목별 ✅적합 / ❌부적합 / ➖해당없음 선택
  5. 📷 사진 선택 버튼 클릭 → 사진 촬영 또는 파일 선택
  6. [점검 완료 저장] 클릭 → 모든 사용자에게 즉시 공유!
  7. [결과 보고서] 탭 → 행 클릭 → 세부 팝업 확인 및 사진 다운로드

🟢 실시간 공유

  • 저장 즉시 모든 사용자 화면에 실시간 반영
  • 사진 → Cloudinary 저장 / 데이터 → Firebase Firestore 저장
  • 새로고침 없이 자동 업데이트

📦 전체 사진 일괄 다운로드 (본청 관리자)

  • [결과 보고서] 탭 하단 → [전체 사진 ZIP 다운로드] 버튼
  • 주소명 폴더로 자동 분류된 ZIP 파일로 다운로드

🟣 중복설치 현장

  • 침수방지시설+차수판 모두 설치된 현장은 🔄 중복 태그 표시
  • 점검표에 두 시설 항목이 통합되어 한 번에 점검 가능

🔴 주의사항

  • 부적합 항목 발견 시 메모란에 사유 필수 기재
  • 사진은 시설 전체가 보이도록 촬영
  • 비 오는 날 스마트폰 방수 처리 후 사용
  • 기관 내부망에서 사진 업로드 안 될 경우 스마트폰 핫스팟 사용
